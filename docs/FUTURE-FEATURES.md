# Future Features & Roadmap

This document outlines planned features for future versions of Talentora. These features are not currently implemented but are on the development roadmap.

## 🚀 Planned Features

### Frontend Job Submission
**Status**: Planned  
**Priority**: High

Allow employers to submit job posts from the frontend without accessing wp-admin.

**Features:**
- Frontend job submission form
- Guest submission with moderation
- Registered user submissions
- Email notifications for new submissions
- Admin approval workflow

---

### Employer Dashboard
**Status**: Planned  
**Priority**: High

A simple frontend dashboard for employers to manage their job listings.

**Features:**
- View all submitted jobs
- Edit job listings
- View application statistics
- Mark jobs as filled
- Renew expired listings

---

### Built-in Application Form
**Status**: Planned  
**Priority**: Medium

Native application form system instead of relying on third-party form plugins.

**Features:**
- Customizable application fields
- Name, email, phone, cover letter
- Resume upload (PDF, DOC, DOCX)
- Custom questions per job
- Application management in admin
- Email notifications

---

### Resume Upload & Management
**Status**: Planned  
**Priority**: Medium

Allow candidates to upload and manage their resumes.

**Features:**
- Resume upload (PDF, DOC, DOCX)
- Resume library for candidates
- One-click apply with saved resume
- Resume preview in admin
- Resume download for employers

---

### Gutenberg Block Support
**Status**: Planned  
**Priority**: Medium

Native Gutenberg blocks for easier job display.

**Features:**
- Job List block
- Single Job block
- Job Search block
- Featured Jobs block
- Recent Jobs block

---

### Smart Filtering & Search
**Status**: Planned  
**Priority**: Medium

Advanced AJAX-powered filtering and search.

**Features:**
- AJAX filtering (no page reload)
- Advanced search filters
- Salary range slider
- Date posted filter
- Remote/On-site filter
- Save search preferences

---

### User Roles & Permissions
**Status**: Planned  
**Priority**: High

Dedicated user roles for employers and candidates.

**Features:**
- Employer role with limited permissions
- Candidate role for job seekers
- Custom capabilities
- Role-based dashboards
- Application tracking per role

---

### Email Notifications
**Status**: Planned  
**Priority**: Medium

Comprehensive email notification system.

**Features:**
- New job posted notifications
- Application received notifications
- Application status updates
- Job expiration reminders
- Customizable email templates

---

### Job Alerts & Subscriptions
**Status**: Planned  
**Priority**: Low

Allow users to subscribe to job alerts.

**Features:**
- Email alerts for new jobs
- Filter-based subscriptions
- Category/type subscriptions
- Location-based alerts
- Frequency settings (daily, weekly)

---
